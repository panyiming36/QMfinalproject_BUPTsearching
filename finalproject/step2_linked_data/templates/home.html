What is Linked Data?

This system publishes BUPT research data as Linked Data. Each resource has a unique URI and can be accessed via HTTP with content negotiation.

Example Resources:

  • Paper: paper_0001
  • Author: author_1234
  • Organization: org_5678

Content Negotiation:

You can get different formats using the Accept header or _format parameter:

curl -H "Accept: text/turtle" http://localhost:8080/research/paper/0001
curl http://localhost:8080/research/paper/0001?_format=json
